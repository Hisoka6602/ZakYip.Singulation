### Motor Card States
1. **Normal State**:
   - Background: White
   - Text: #0F172A (Dark)
   - Border: Subtle shadow

2. **Selected State**:
   - Background: #3B82F6 (Primary blue)
   - Text: White
   - Border: None
   - Shadow: Enhanced

3. **Abnormal State**:
   - Background: #EF4444 (Red)
   - Text: White
   - Border: None

4. **Disabled State**:
   - Background: Transparent
   - Text: #94A3B8 (Gray)
   - Border: #CBD5E1 (Light gray)

## Typography

- **Page Title**: 28pt, Bold
- **Button Labels**: 14pt, Bold
- **Motor ID (M01-M20)**: 14pt, Bold
- **Motor RPM**: 32pt, Bold
- **Motor Unit**: 12pt, Regular
- **Batch Info**: 14pt, Regular
- **Mode Switcher**: 15pt, Bold

## Spacing & Sizing

- **Card Border Radius**: 24dp
- **Button Border Radius**: 20dp
- **Icon Button**: 44x44dp
- **Toolbar Button Height**: 56dp
- **Main Action Button**: 60dp height
- **Grid Spacing**: 8dp between cards
- **Section Spacing**: 16dp between sections
- **Padding**: 16dp page padding

## Shadows

All cards and elevated elements use soft shadows:
- Offset: (0, 2)
- Blur Radius: 8-12dp
- Opacity: 0.06-0.08
- Color: Black

## Interactions

1. **Tap on Motor Card**: 
   - Card background changes to #3B82F6
   - Text color changes to white
   - Previous selected card returns to normal state

2. **Tap on Safety Command Button**:

3. **Tap on Axis Speed Setting Button**:

   - Active button has primary blue background
   - Inactive button has transparent background

5. **Tap on Main Action Button**:
   - Shows confirmation dialog with batch info

## Responsive Behavior

- Content is scrollable vertically
- Grid maintains 3 columns on all screen sizes
- Buttons in toolbars distribute evenly
- Safe area insets are respected (iOS)

## Animations

- Button taps: Scale feedback (0.95)
- Mode switcher: Smooth transition (200ms)
- Card selection: Fade-in/out (150ms)
- Dialogs: Slide up from bottom (iOS style)

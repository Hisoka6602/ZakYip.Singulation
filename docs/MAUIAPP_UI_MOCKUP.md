## 图标尺寸规范

| 位置 | 图标大小 | 文字大小 | 说明 |
|-----|---------|---------|------|
| 页面主标题 | 32px | Headline 样式 | 最大最显眼 |
| 区域标题 | 24px | 20px Bold | 次级标题 |
| 按钮图标 | 20px | 16px | 按钮内图标 |
| 列表项图标 | 14-16px | 14-16px | 列表内容 |
| 小图标 | 14px | 12-14px | 辅助信息 |
| 空状态图标 | 48px | 16px | 占位符 |

## 间距规范

- 页面边距：20px
- 区域间距：15px
- 元素间距：10px
- 图标与文字间距：5-8px
- 按钮内边距：15px (横向), 10px (纵向)

## 交互说明

1. **自动发现**：进入设置页面后 500ms 自动开始 UDP 服务发现
2. **服务选择**：点击发现的服务卡片，自动填充 API 地址
3. **状态反馈**：所有操作都有明确的状态消息反馈
4. **加载状态**：操作进行中显示加载动画，按钮禁用
5. **空状态**：列表为空时显示友好的空状态提示

## 响应式布局

- **小屏幕 (< 400px)**：单列布局，按钮全宽
- **中等屏幕 (400-800px)**：双列按钮布局（使能/禁用）
- **大屏幕 (> 800px)**：保持中等屏幕布局，增加左右边距
